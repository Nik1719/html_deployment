Purpose of This Project

This project demonstrates how modern frontend applications are deployed on AWS using industry-standard DevOps practices. It focuses on static website hosting, scalability, security, and automation.

Technology Stack

Frontend

  • HTML & CSS
  • Responsive UI design
  • Optimized static assets

CI/CD Pipeline

  • GitHub Actions
  • Automated build & deployment
  • Secure AWS credentials via secrets

AWS Cloud Services

  • Amazon S3 (Static Website Hosting)
  • CloudFront (Global CDN)
  • IAM (Secure access control)

Deployment Workflow

01

Application Development

Static frontend developed with clean and optimized assets.

02

CI/CD Automation

03

AWS Deployment

Amazon S3 hosts the static website, while CloudFront delivers content securely with low latency worldwide.
